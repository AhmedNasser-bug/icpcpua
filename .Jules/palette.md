## 2024-04-23 - Accessible Neobrutalism
**Learning:** High-contrast Neobrutalist design (thick black borders, heavy shadows, intense background colors) can make standard focus rings (`focus-visible`) invisible or muddy if the ring color clashes or blends with the border/shadow. Additionally, using standard `<a>` tags wrapping `<button>` elements for massive CTAs can cause redundant focus states and screen reader confusion if not handled correctly.
**Action:** Always enforce a high-contrast focus ring (`focus-visible:ring-4 focus-visible:ring-black focus-visible:outline-none`) on interactive elements, ensuring it stands out against the vibrant background colors. When wrapping buttons in links for design purposes, apply the `aria-label` to the `<a>` tag and add `tabIndex={-1}` to the inner `<button>` to prevent double-focusing while preserving the semantic link behavior and the desired button aesthetic. Added these patterns to `<HeavyCTAButton>` and the Back button.
